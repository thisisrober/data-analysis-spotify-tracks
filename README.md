# Spotify Tracks Dataset — Audio Features Analysis

Comprehensive exploratory data analysis of the Spotify Tracks Dataset with audio features. This project analyzes musical characteristics, patterns, and trends across thousands of Spotify tracks.

Quick start:

1. Create and activate a Python environment.
2. Install dependencies: `pip install -r requirements.txt`.
3. Open the notebook: `notebooks/analysis-es.ipynb` or `notebooks/analysis-en.ipynb`.
